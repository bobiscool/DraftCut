# DraftCut Storyboard — User Guide

> Screenshot this page from: `http://localhost:8792/?lang=en&doc=1`  
> (`lang=en` = English UI · `doc=1` = hides clip card list, footer, progress panel)

## What you see

![Storyboard overview](screenshots/storyboard-overview.png)

- **Player** — preview clips in edit order  
- **Timeline** — multi-track hand-drawn plan (Main, Story, Subplot, Music, Emotion)  
- **Filmstrip** — thumbnails on each clip  
- **Zoom** — slider or Ctrl/Cmd + scroll on timeline  

## Workflow

1. `node scripts/run.mjs <folder> --brief "…" --duration 60 --open`
2. Watch live progress (Scan → Transcribe → Analyze → Sequence)
3. Tweak on timeline → **Save** → **Export FCPXML / CapCut**
