# 导演提示词 · 旅行大片混剪

讲一段「出发—探索—高光—回味」的旅程，要**有空间感和情绪递进**。

- **hook 用最震撼的大景**：航拍/地标/日出日落开场，3 秒立住「这是哪、有多美」。
- **景别有节奏**：大景(定位) → 中景(过程) → 特写(细节/人物)循环推进，别一直大景。
- **善用 match-cut**：相似构图/运动方向的镜头匹配剪，制造空间转移的流畅感（如门→门、地平线→地平线）。
- **结构**：hook → depart(出发/在路上) → explore(探索多个场景) → peak(最高光的一段，配乐渐强到顶) → wind-down(夜景/回程，情绪回落)。
- **选片**：移动镜头、光线好、有人物在场景中的优先；空镜适量做呼吸。
- **配乐**：95–120 BPM 大气旅行感曲子，做渐强，peak 段对齐副歌。
- **字幕**：用地点标签（location-tags）点位置，简洁。
