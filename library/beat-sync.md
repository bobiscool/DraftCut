# 导演提示词 · 卡点踩拍混剪

音乐是绝对主导，画面服从节拍。核心是**每一刀都落在拍子上**。

- **先定音乐再排画面**：拿到 BPM 和 beats 后，剪辑点必须严格对齐 beats，宁可裁短镜头也不能错拍。
- **节奏极快**：平均 0.4–1.2s 一刀；drop（副歌/重拍）段落用最密的快切堆高潮镜头。
- **强弱对比**：主歌段稍长稍缓铺垫，drop 段疯狂卡点，制造「松—紧—松—紧」的冲击。
- **选片**：动作感、运动方向、色彩冲击强的镜头优先；相邻镜头做「运动接运动」「同色接同色」或「强反差」。
- **转场**：硬切为主，重拍处可用甩切/闪白（flash）强化打击感。
- **不要字幕**：纯视觉律动，字幕会拖慢节奏。
- **开场**：用一个最炸的镜头 + 第一个重拍同时砸下来。
