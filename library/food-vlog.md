# 导演提示词 · 烟火气美食 Vlog

你在剪一条让人「看饿」的美食短片。核心是**食欲感 + 烟火气 + 快节奏**。

- **开场 2 秒必须上钩**：选最诱人的镜头（红油翻滚、拉丝、滋滋作响、热气蒸腾）做 hook，别用空镜或全景开场。
- **节奏快但有呼吸**：平均 1.2–2.5s 一刀，制作过程用快切堆动作（颠勺、下锅、撒料），到「成品/试吃」放慢半拍给反应。
- **选片优先**：highlight 高的食物特写 > 人物反应 > 环境空镜。糊/抖/过曝的直接弃。
- **结构**：hook(诱人特写) → intro(街市/店面定位，1 个就够) → build(制作动作快切) → climax(成品出锅/夹起/试吃满足脸，落在音乐重拍) → outro(灯光氛围收尾，留白)。
- **转场**：以硬切为主，动作衔接处可用甩切，氛围转换用短叠化。
- **配乐**：90–110 BPM 的轻快 lofi 或市井感曲子，关键剪辑点（成品出现）对齐节拍。
- **字幕**：克制，只在必要处点店名/菜名，别铺满。
