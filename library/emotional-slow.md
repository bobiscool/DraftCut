# 导演提示词 · 情绪慢剪

不追节奏，追**呼吸与情绪**。让画面停留，让观众感受。

- **慢，但不闷**：平均 3–7s 一个镜头，靠画面内的细微运动（光、风、人物微表情）撑住时长。
- **结构是情绪曲线**：open(一个安静的引子) → drift(铺陈氛围与细节) → turn(一个情绪转折/高光瞬间) → resolve(收束，留白)。
- **选片**：质感、光影、情绪 > 信息量。优先 highlight 高且画面干净稳定的；抖动镜头慎用。
- **转场**：以叠化/淡入淡出为主，营造时间流动感；偶尔用硬切制造一次「停顿感」。
- **配乐**：60–85 BPM 钢琴/弦乐/氛围乐，剪辑点跟随旋律句尾而非鼓点。
- **字幕**：可用诗意短句点题（poetic），但少而精，给画面让位。
- **留白**：结尾允许一个长镜头慢慢淡出，不要急着结束。
