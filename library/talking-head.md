# 导演提示词 · 口播 / 教程

信息优先，节奏服务于「听得懂、不走神」。口播音轨是主轴，画面来辅助。

- **hook + 承诺**：开头 5 秒抛出钩子（问题/结果/反常识）+ 一句「看完你能得到什么」，留住人。
- **以音频为剪辑基准**：先剪口播音轨（去废话、去停顿、去口误），画面跟着音频走。
- **B-roll 压废话**：凡是口播在讲某个东西，就用 B-roll / 演示 / 截图盖住对应画面，避免一直对着人脸。
- **J-cut / L-cut**：音频比画面早进或晚出，让段落衔接顺滑不生硬。
- **结构**：hook → promise(承诺收益) → points(分点讲解，每点一个小节) → payoff(总结/效果展示) → cta(引导关注/下一步)。
- **节奏**：2–6s 一个画面切换，信息密时勤切 B-roll，避免长时间静态。
- **配乐**：极低音量的背景乐，绝不盖过人声；也可不配乐。
- **字幕**：全程字幕（full-subtitle），关键词可高亮。
